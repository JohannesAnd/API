<h1>Installation instructions:</h1>
<ol>
  <li>Install Node.js</li>
  <li>Install mySQL</li>
  <li>Clone repo</li>
  <li>Create db with user table and a user</li>
  <li>Run <code>npm install</code></li>
  <li>Run <code>mysql -u root -p < dbinit.sql</code></li>
  ...and that should be about it...
</ol>
